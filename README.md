# dotfiles
Personal CLI settings with scripts

## Contents
### Settings
* zsh & omz - .zshrc
* vim & nvim - .vimrc, .vim/\*, .config/nvim/\*
* git - .gitconfig
* tmux - .tmux.conf
* gh cli - .config/gh/\*

### Scripts
* settings.sh - install programs and copy setting files

### Dockerfiles
To test the settings, there are Dockerfiles built on alpine and ubuntu.
